# v0.2.2

Adds configurable tool disabling, Exa output trimming, full-page fetch support, and safer pagination behavior for **pi-search**.

## Added

- `web_fetch` tool for reading a full page as markdown after `websearch` or `codesearch`
- `~/.pi/agent/pi-search.json` support for disabling specific tools
- offset pagination for long `context7` and `web_fetch` responses

## Improved

- truncated Exa search highlights to reduce context waste in `websearch` and `codesearch`
- extracted shared helpers for pagination and error handling
- wired `AbortSignal` through `grepsearch`

## Fixed

- prevented malformed `disabledTools` config values from crashing extension startup
- clamped pagination offsets and enforced non-negative offset schemas
- stopped `context7` continuation pages from repeating the document header

## Included tools

- `websearch` — web search via Exa AI
- `codesearch` — code/documentation search via Exa AI
- `context7` — official documentation lookup
- `grepsearch` — real-world GitHub code search via grep.app
- `web_fetch` — full page fetch via Exa AI
