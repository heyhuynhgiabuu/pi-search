# v0.2.0

Adds documentation lookup and improves package positioning for **pi-search**.

## Added

- `context7` tool for library documentation lookup
  - `resolve` operation to find a library ID
  - `query` operation to fetch docs for a topic

## Updated

- improved `README.md` introduction and usage guidance
- clearer package metadata and keywords in `package.json`
- `pi-search` now presents itself as a unified research toolkit for pi

## Included tools

- `websearch` — web search via Exa AI
- `codesearch` — code/documentation search via Exa AI
- `context7` — official documentation lookup
- `grepsearch` — real-world GitHub code search via grep.app
