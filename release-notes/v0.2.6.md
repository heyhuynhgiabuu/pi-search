# pi-search v0.2.6

## Highlights

- **context7 works again** on the `type=txt` docs endpoint (no more JSON parse errors on `###` Markdown).
- **Full-width tool panels** for Markdown tools: `Box` + `toolSuccessBg`, with background preserved across Markdown ANSI color resets.
- **Collapsed hints** use lowercase `ctrl+o to expand` (Pi-consistent).

## Fixed

- `context7` `fetchDocs()` — read response as text; JSON only when body/content-type is actually JSON.
- `renderResult` for `websearch`, `web_fetch`, `context7`, `deepwiki` — `preserveBoxBackground()` so colored titles/links do not strip the panel background mid-line.

## Changed

- Markdown tool results use `theme.bg("toolSuccessBg")` inside `Box` for terminal-width padding.
- Collapsed preview remains ~1.5k chars with `ctrl+o to expand`.

## Upgrade

```bash
pi install npm:@heyhuynhgiabuu/pi-search
# or
pi update --extensions
```

Requires Pi **0.80.x** (`@mariozechner/pi-coding-agent` peer).
